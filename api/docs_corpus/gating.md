body: |
  ConnectWork's policy gate makes approval readiness a deterministic engine decision rather than a model preference. It evaluates required signoffs, delegated authority thresholds, and evidence completeness before any generated answer or action can claim a decision is ready. When a gate blocks, the assistant may explain the reason and route the work to the right approver, but it cannot override the block. The live gating page provides the full implementation narrative and user-facing examples.
